# Rohit Sahani Portfolio

Premium static portfolio website built with HTML, CSS, and vanilla JavaScript for GitHub Pages deployment.

## File Structure

```text
Portfolio/
|-- assets/
|   |-- icons/
|   |   `-- favicon.svg
|   `-- images/
|       |-- og-cover.svg
|       |-- profile-placeholder.svg
|       |-- project-atlas.svg
|       |-- project-creator.svg
|       `-- project-trails.svg
|-- index.html
|-- script.js
|-- style.css
`-- README.md
```

## Customize Content

1. Open `index.html`.
2. Replace the text inside the hero, about, skills, projects, experience, and contact sections.
3. Update profile and project images inside `assets/images/`.
4. Replace email, phone number, and social/profile links with your own.
5. Update the animated roles array inside `script.js` if you want different hero typing text.

## Upload to GitHub

1. Create a new repository on GitHub.
2. Upload all files and folders from this project to the repository root.
3. Commit and push the files.

## Enable GitHub Pages

1. Open your repository on GitHub.
2. Go to `Settings` > `Pages`.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Select the `main` branch and `/ (root)` folder.
5. Save the settings and wait for GitHub to publish the site.

## Notes

- This site is fully static and does not require a backend.
- The contact form uses `mailto:` and opens the visitor's email client.
- All local assets use relative paths, so the project works on GitHub Pages.
